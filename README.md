## Sobre
Formulário de matrícula estático desenvolvido em HTML e CSS para a escola de educação infantil Estrelas do Amanhã. O projeto apresenta um formulário com campos para informações da criança, endereço residencial, dados dos responsáveis, histórico médico e upload de documentos obrigatórios.
       
## 🚀 Tecnologias
- HTML5
- CSS3
- Google Fonts (importadas em `index.html`)

## Como executar
1. Clone este repositório ou copie os arquivos para sua máquina.
2. Abra o arquivo `index.html` no navegador.
3. Opcional: use Live Server no VS Code para visualizar localmente.

## Estrutura das pastas 
  - `index.html` - Formulário de matrícula
  - `styles/` - Arquivos de estilo CSS
  - `global.css` - Estilos base e variáveis
  - `index.css` - Importações e configuração geral
  - `layout.css` - Estilos de layout
  - `forms.css` - Estilos dos formulários
  - `fields/` - Estilos específicos dos campos
  - `buttons.css` - Estilos de botões
  - `checkbox.css` - Estilos de checkboxes
  - `input.css` - Estilos de inputs
  - `radio.css` - Estilos de radio buttons
  - `droparea.css` - Estilos da área de upload
  - `assets/` - Recursos visuais (ícones e imagens)
  - `icons/` - Ícones do projeto
  - `image/` - Imagens do projeto

## Funcionalidades
- Formulário completo com validação de campos
- Seção de Informações da Criança (nome, data de nascimento, sexo, informações médicas)
- Upload de certificado de nascimento via drag-and-drop
- Seção de Endereço Residencial (CEP, rua, número, cidade, estado)
- Seção de Informações dos Responsáveis (dados e contato)
- Estilos visuais consistentes e agradáveis
